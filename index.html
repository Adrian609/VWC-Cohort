Understanding grid-template-areas

By exploring the CodePen example below, you can see how the width and height of each grid item is determined.


There are two columns. The first column has a width of 1fr. The second column has a width of 2fr. When added together the width totals three fractions, meaning the first column occupies 1/3rd of the width while the second column occupies the remaining 2/3rds of the width.
